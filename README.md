# 📊 Finance Dashboard in Excel

## About the Project

This project is a Finance Dashboard built in Microsoft Excel to analyze revenue, expenses, and profit using interactive charts and slicers. The dashboard provides a simple way to track financial performance and compare different business metrics.

It allows users to filter data by year, month, quarter, department, and payment method, making it easier to explore the data and gain useful insights.

## Features

- Interactive Excel dashboard
- Revenue, Expense, and Profit analysis
- Monthly profit trend
- Revenue and Expense by category
- Department-wise Revenue vs Expense comparison
- Dynamic slicers for easy filtering
- Clean and user-friendly layout
